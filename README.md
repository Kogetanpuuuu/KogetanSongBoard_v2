# Kogetan SongBoard v1 Firebase

## ファイル
- view.html: TikTok LIVE Studioに貼る表示画面
- control.html: 操作用
- firebase-config.js: Firebase設定を貼る場所

## 手順の概要
1. FirebaseでRealtime Databaseを作る
2. firebase-config.jsの firebaseConfig を置き換える
3. GitHubに全ファイルをアップロード
4. Vercelで公開
5. TikTok LIVE Studioのリンクに /view.html を貼る
6. 操作は /control.html を開く
